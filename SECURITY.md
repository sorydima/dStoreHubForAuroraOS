# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.0.x   | ✅         |
| < 1.0   | ❌         |

## Reporting a Vulnerability

Please report security vulnerabilities to [security@rechain.network](mailto:security@rechain.network).

We will respond within 72 hours and coordinate a fix with you.
